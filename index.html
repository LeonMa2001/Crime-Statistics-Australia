Healthsites Of Australia

Healthsite Mapped on Australia

This map visualisation initially illustrates all of the accumulated data points of healthsites listed in the dataset. It plots it onto the Australian map which also includes filters to be able to search for specific healthsite types.

Need for healthsites per state

Left Div

This is a graph that indicate the percentage of conditions, each state's residents hold. The graph indicates the percentage of the total population's health conditions and compares it with each other state as well as the country average.

Right Div

This is a graph that indicate the percentage of health factors, each state's residents inhibit. The graph indicates the percentage of the total population's health conditions in reference to a breach in health factors, whether it be a specific factor or a health behavioural recommendation that has been breached.
